/// HOW IT WORKS

  1. Closed category enum. The system prompt forces one of six labels — no free-text drift breaking downstream routing.
  2. Self-rated confidence. The model is told to lower its confidence on ambiguous inputs, so a 0.6 score is a real signal to escalate.
  3. No-invented-facts rule. The prompt forbids fabricating lot numbers, dates, or staff names — staff can edit the draft without quietly correcting hallucinations.
  4. Three flags. contains_pii, ambiguous_input, needs_human_review, out_of_scope — downstream automation can mask, encrypt, or escalate.
  5. Draft-not-send. Every output is a draft a human reads before it goes out. There is no autonomous reply mode.
